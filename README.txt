Получение списка книг, GET:
http://127.0.0.1:8000/api/books/

Получение списка книжных магазинов, GET:
http://127.0.0.1:8000/api/bookshops/

Фильтрация книжных магазинов по названию, GET:
http://127.0.0.1:8000/api/bookshops/?city=Ульяновск

Фильтрация книг по названию, GET:
http://127.0.0.1:8000/api/books/?title=1984

Фильтрация книг по конкретного издательству, GET:
http://127.0.0.1:8000/api/books/?publishing_house=АСТ

Фильтрация по автору, GET:
http://127.0.0.1:8000/api/books/?author=Оруэлл
